# Boss Daddy v2 — Full-Codebase Audit

> **Date:** 2026-07-19
> **Method:** 20-agent parallel sweep (10 domains) + adversarial verification pass. Every finding was re-checked against live code by an independent skeptic agent; findings not backed by real code were refuted or downgraded.
> **Baseline:** working tree of `brand-v3.4-messaging` (architecturally equivalent to `master`; v3.4 is a brand/messaging layer).
> **Lens:** solo operator using AI assistance now, recruiting/growing a team later. Recommendations favor guardrails, chokepoints, tests, and cheap onboarding — **not** enterprise-scale infra.
> **Overall health: ≈6.5/10** — strong fundamentals and unusually mature plumbing for a solo build, undermined by three cheap-to-fix Critical operational gaps and a dependency-currency blind spot.

---

## The one theme that matters

**The money path fails *silently*.** Multiple independent systems on the revenue path lose money or hide errors — with zero alert — when a single env var or dependency is wrong (Amazon tag unset → total commission loss; caught webhook errors never reach Sentry; rate-limit "budget guard" fails open on missing Upstash). Fixing the silence is higher-leverage than any single bug.

### 🟡 Medium

- **Comment moderation trusts unvalidated AI JSON** — malformed shape silently auto-publishes (`app/api/comments/route.ts:71-159`); the sibling `moderate` route validates with Zod. Reuse that schema.
- **AI duplication** — error-classification regex ×5 routes; 3 divergent JSON-extraction strategies; `pause_turn` web_search loop ×2. → `lib/claude/errors.ts` + `parseModelJson()` + `runWebSearchTurn()`.
- **AI latency** — 4 generation routes hold HTTP open up to 180s; the correct `createJob/after()/poll` pattern exists but serves only `specs-grade`. Widen it.
- **Rate limiter fails open** if Upstash env missing — incl. the radar cron's "non-negotiable budget guard" (`lib/rate-limit.ts:81`). Log + fail-closed for the dollar-cap cron.
- **No backup/PITR runbook**, yet a hard-delete cron on real user data (`delete-expired-accounts:64`) touching kids'/DM/financial data. Confirm PITR *enabled*; write a restore runbook.
- **No feature-flag mechanism** (`lib/flags.ts` absent) despite half-built surfaces on auto-deploy-from-master.
- **14 of 18 admin routes hand-roll their own admin check** instead of `requireAdminApi()` — one missed copy = one open admin route.
- **`(admin as any)` casts** on the orders table are now unnecessary dead code (types exist) — highest-stakes file.
- **Stripe SDK 5 majors behind** (`^17.7.0` vs 22.x) on the money path.
- **A2 — PostgREST `.or()` filter injection** (`app/api/members/search/route.ts:20`): raw `q` interpolated into `.or()` → craft `,id.neq.<uuid>` to dump the full member directory. The exact fix already exists on the public search page (`app/(public)/search/page.tsx:41`) but wasn't propagated. Extract to `lib/postgrest-escape.ts`, apply everywhere. (CWE-943)
- **A3 — Printful webhook uses a static query-string token, not a signature** — secret lands in logs; forged `package_shipped` events overwrite any order's tracking. Move to header, rotate, rate-limit.
- **A11y** — `--bd-text-faint #71717a` = 3.67:1 on dark surface, fails WCAG AA (needs 4.5:1), used on ~15 components at 12px. Lighten to zinc-400.
- **Perf** — homepage ships both hero `<Image>`s always-mounted (desktop preloads mobile hero); desktop LCP image lacks `priority`.

## What's genuinely excellent (verified — keep doing this)

- **RLS "admin is moderation-only" doctrine** actively enforced (migs 106–108), not just documented; DM attachment proxy is textbook defense-in-depth (RLS-scoped read → short-lived signed URL, private bucket).
- **`proxy.ts`** — thin 51-line orchestrator delegating to 6 single-responsibility modules with documented ordering. The chokepoint pattern done right.
- **`createStructured()`** tool-forcing eliminates the malformed-JSON failure class; prompt caching ordered correctly per Anthropic guidance; Boss agent Haiku-first cost routing.
- **`lib/sanitize.ts`** — tight allowlist, `javascript:` blocked, `rel` merge for affiliate anchors, applied in the correct order (resolve tokens → sanitize) on every HTML UGC write path.
- **Secrets hygiene clean** — every `NEXT_PUBLIC_*` is legitimately public; the 3 real secrets each live in exactly one server-only file.
- **Money path integrity** — Stripe webhook HMAC verified before logic + idempotency guard; checkout prices server-derived (no client tampering); SSRF-guarded `/api/img` + `/api/og`; image uploads validated via sharp + EXIF stripped.
- **Strict TS, zero `@ts-ignore`**, disciplined `LABELS.*` naming, incident-derived guardrails (`check-migrations.yml`, middleware-convention + OG-coverage checks).
